# FAR-MARS

## Control Software

### Building

#### Linux:

In FAR-MARS/controlsoftware, run `make`

#### Windows:

From a linux platform, download `mxe` and compile for qt5

Then run:

`{$mxe_dir}/bin/i686-w64-mingw32.static-qmake-qt5`

followed by `make`

The windows EXE will be in the new `release` directory
